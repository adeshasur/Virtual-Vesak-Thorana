# Budu Siritha Digital Vesak Thorana

Budu Siritha Digital Vesak Thorana is an interactive one-page web experience that brings a traditional Sri Lankan Vesak thorana into a modern browser-based format with respectful Sinhala story panels.

## Features

- Animated Vesak thorana with glowing lights
- Seven clickable Sinhala story panels from the life of the Buddha
- Popup story viewer with previous and next controls
- User-triggered peaceful background ambience
- Central respectful Buddha silhouette illustration
- Floating lanterns, night background, glowing bulbs, and responsive layout
- Plain HTML, CSS, and JavaScript

## Files

- `index.html` - page structure and thorana panels
- `style.css` - Vesak theme, layout, responsive styling, and animations
- `script.js` - story modal logic and music toggle
- `assets/images/vesak-thorana-background.png` - generated visual background asset
- `assets/images/buddha-silhouette.png` - generated central symbolic illustration
- `assets/images/buddha-statue-center.png` - generated realistic central statue-style illustration
- `assets/images/realistic-thorana-stage.png` - generated full-stage thorana render used as the main interactive visual
- `assets/images/realistic-thorana-stage-transparent.png` - transparent-background version used on the page
- `assets/images/realistic-thorana-stage-v2.png` - improved full-stage thorana render with clearer statue and panels
- `assets/images/realistic-thorana-stage-v2-transparent.png` - transparent version of the improved render used on the page
- `assets/images/story-*.png` - generated illustrations for the seven story panels
- `assets/audio/` - optional music playlist folder. Add licensed MP3 files named `buddhist-instrumental.mp3`, `temple-bells.mp3`, and `pirith-chant.mp3`; the app falls back to generated temple ambience when files are missing.

## Run

Open `index.html` in a browser, or serve the folder with any static web server.
